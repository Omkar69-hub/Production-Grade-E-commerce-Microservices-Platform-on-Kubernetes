# Resume Snippet

**Cloud-Native E-Commerce Platform** | *Personal Project*
*   Architected and developed a production-grade microservices e-commerce platform using **Java Spring Boot**, **React**, **PostgreSQL**, **Redis**, and **RabbitMQ**.
*   Containerized all services with multi-stage Docker builds and orchestrated deployment on **AWS EKS** using **Kubernetes** and **Helm**.
*   Provisioned highly available AWS infrastructure (VPC, RDS, ElastiCache) using **Terraform**, ensuring environment parity across dev/prod.
*   Implemented a strict GitOps CI/CD pipeline leveraging **GitHub Actions** and **Argo CD** for automated, secure, and traceable deployments.
*   Hardened cluster security by enforcing **RBAC**, Network Policies, Pod Security Standards, and integrating **Trivy** and **Checkov** for vulnerability scanning.
