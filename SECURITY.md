# Security Policy

## Supported Versions
Security updates are provided for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to [security@ecommerce.local]. You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity.

We use Trivy, Checkov, and Gitleaks in our CI pipeline to catch the majority of vulnerabilities before they are merged.
